# AmongUrbis
Game with the theme of sustainable urban development.

Jogo se passa dentro de uma prefeitura que mais parece um labirinto, em alguns cômodos existem objetos que ao clicados pelo mouse iniciam mini-tarefas para ajudar a cidade a ser mais sustentável. As Tarefas estão escondidas, faz parte do jogo procura-las.

- Os três desafios são:
-- Coletar os dejetos de um tanque de líquidos para tratamento da água. (Representado por um cubo azul com esferas brancas que ao serem clicadas somem, para resolver a task basta retirar todos os dejetos).

-- Reduzir os poluentes lançados na atmosfera pela industria, clicar várias vezes nas torres diminuem até paralizar totalmente a emissão dos gases.(Essa task está representada por dois cilindros e um cubo com emissão de párticulas, uma vez que todos os gases foram cessados a tarefa estará concluída).

-- Depositar o lixo no lugar correto para a reciclagem, existem alguns tipos de lixo no chão, seu trabalho é coleta-los e coloca-los no lugar correto, basta clicar e arrastar os dejetos para a caixa da cor correta. Mas tome cuidado, caso erre o lixo de lugar, terá que voltar para o início e jogar tudo de novo!!!(Representado por cubos com as cores respectivas dos cubos maiores que são as latas de lixo). 

A ideia inicial era fazer uma série muito maior de minijogos abordando a temática como um todo. Além de adicionar um sistema de seleção arbitrária das eventuais tasks a serem feitas na rodada. 

As dificuldades encontradas foram pelo tempo, acabei levando dois dias pra estruturar a ideia do jogo, inspirada no Among Us.
